# ash - a shell

**ash** is a simple shell in C.

## installing

1. **clone the repository**

   ```git clone https://www.github.com/larcoverde/ash```

2. **compile and install**

   ```cd ash```
   
   ```sudo make install```

3. **run**

   ```ash```
